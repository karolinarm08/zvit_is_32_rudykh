У ході виконання роботи було закріплено навички роботи з Web Storage API для збереження даних у локальному сховищі браузера, створено та використано модульний код для забезпечення структурованості проекту. Застосовано принципи делегування подій, що дозволяє ефективно працювати з великою кількістю елементів у DOM.
Було створено макет галереї довільних зображень, що дозволило повторити принципи верстки, стилізації, а також динамічного створення елементів через JavaScript. За допомогою бібліотеки basicLightbox реалізовано функціональне модальне вікно, яке відкриває великі версії зображень із галереї, забезпечуючи зручний інтерфейс користувача.
